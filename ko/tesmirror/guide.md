## TesMirror란?

TesMirror는 안드로이드 스마트폰 화면을 테슬라 차량의 브라우저에 실시간으로 미러링해주는 앱입니다.

**주요 특징:**
- 루팅 없이 사용 가능
- 테슬라 화면 터치로 폰 제어
- 간편한 원터치 미러링 시작
- 약 3분 만에 초기 설정 완료

<img src="../../images/guide/01_main_screen.png" alt="TesMirror 메인 화면" width="50%">

*TesMirror 앱 메인 화면 - 미러링 실행 버튼과 테더링 설정 바로가기*

---

## 준비물

미러링을 시작하기 전에 다음 사항을 확인하세요:

- **스마트폰**: Android 10 이상
- **테슬라 차량**: Model S, Model 3, Model X, Model Y
- **네트워크**: 폰의 핫스팟(테더링) 기능

> **중요:** 테슬라 차량이 스마트폰의 핫스팟에 연결되어 있어야 합니다.

---

## 초기 설정

TesMirror를 처음 사용할 때 간단한 초기 설정이 필요합니다. 약 3분이면 완료됩니다.

### 1단계: 앱 설치

Google Play 스토어에서 **TesMirror**를 검색하여 설치합니다.

### 2단계: 권한 허용

앱 실행 시 요청하는 권한들을 허용해주세요:
- 화면 녹화 권한 (미러링용)
- 접근성 서비스 권한 (터치 제어용)
- 알림 권한
- 배터리 최적화 제외 권한 (백그라운드에서 미러링 중단 없이 사용)

<img src="../../images/guide/02_setup.png" alt="초기 설정 단계" width="100%">

*초기 설정 5단계: 알림 권한 → 원격 제어 권한 → 접근성 설정 → 배터리 최적화 제외 → 미러링 허용*

### 3단계: 설정 완료

모든 권한을 허용하면 설정이 완료됩니다.

<img src="../../images/guide/03_setup_complete.png" alt="설정 완료" width="50%">

*설정 완료 화면 - 사용 방법이 간단히 안내됩니다*

## 사용 방법 (4단계)

초기 설정이 완료되면, 다음 4단계로 미러링을 시작할 수 있습니다.

### Step 1: 테더링 연결

1. 스마트폰의 **핫스팟(테더링)**을 켭니다
2. 테슬라 차량의 Wi-Fi 설정에서 스마트폰 핫스팟에 연결합니다

<img src="../../images/guide/04_tethering.jpg" alt="테더링 설정" width="50%">

*핫스팟을 켜고 테슬라를 연결하세요*

> **팁:** 앱의 "테더링 설정 가기" 버튼을 누르면 핫스팟 설정으로 바로 이동합니다.

---

### Step 2: 미러링 시작

TesMirror 앱을 열고 **미러링 실행** 버튼을 탭합니다.

<img src="../../images/guide/01_main_screen.png" alt="미러링 시작 버튼" width="50%">

*앱에서 "미러링 실행" 버튼을 탭하세요*

버튼을 누르면 미러링 서버가 시작되고, 연결 주소가 화면에 표시됩니다.

---

### Step 3: 테슬라 브라우저에서 주소 입력

1. 테슬라 화면에서 **브라우저**를 엽니다
2. 앱에 표시된 주소(예: `http://10.0.2.16:9761`)를 입력합니다
3. **이동** 버튼을 탭합니다

<img src="../../images/guide/06_connection_address.png" alt="연결 주소 표시" width="50%">

*서버 실행 중 - 표시된 주소를 테슬라 브라우저에 입력하세요*

---

### Step 4: 미러링 시작!

연결이 완료되면 스마트폰 화면이 테슬라 대형 스크린에 표시됩니다.

<img src="../../images/guide/07_tesla_mirroring.png" alt="테슬라에서 미러링된 화면" width="100%">

*테슬라 브라우저에 미러링된 스마트폰 화면*

이제 테슬라 화면을 터치하여 스마트폰을 제어할 수 있습니다!

## 주요 기능

### 1. 원터치 미러링

버튼 하나로 간편하게 미러링을 시작합니다.

### 2. 터치 컨트롤

테슬라 터치스크린으로 스마트폰을 직접 제어할 수 있습니다.

**지원하는 터치 동작:**
- **탭**: 화면 터치
- **스와이프**: 위/아래/좌/우 스크롤
- **핀치 줌**: 두 손가락으로 확대/축소

### 3. 대화면 네비게이션

테슬라의 15~17인치 대형 스크린에서 네비게이션 앱을 더 크고 선명하게 볼 수 있습니다.

## 자주 묻는 질문 (FAQ)

### Q1: 연결이 안 될 때는 어떻게 하나요?

**A:** 다음 사항을 확인해주세요:
1. 테슬라가 스마트폰 핫스팟에 연결되어 있는지 확인
2. 핫스팟이 활성화되어 있는지 확인
3. 앱을 재시작하고 다시 시도

### Q2: 화면이 끊기거나 느릴 때는?

**A:** 핫스팟 신호가 약할 수 있습니다:
1. 스마트폰을 테슬라 화면 가까이 두세요
2. 다른 기기의 핫스팟 연결을 해제하세요

### Q3: 터치가 작동하지 않아요

**A:** 접근성 서비스 권한이 활성화되어 있는지 확인하세요:
1. 스마트폰 설정 > 접근성
2. TesMirror 찾기
3. 서비스 활성화

### Q4: 배터리 소모가 많은가요?

**A:** 화면 미러링 특성상 배터리 사용량이 증가합니다. 장시간 사용 시 차량 충전기 연결을 권장합니다.

### Q5: 루팅이 필요한가요?

**A:** 아니요, TesMirror는 **루팅 없이** 사용 가능합니다.
